# 片段属性

## 基本信息
- 片段名称 【单行文本】 
- 片段描述 【多行文本】

## 风格设定
- 风格 【选项】
- 情绪氛围 【选项】

## 角色与场景
- 角色 【多行文本】
- 场景设定 【单行文本】

## 视频参数
- 画幅比例 【选项】
- 视频时长（秒）

## 声音设计
- 配乐风格 【选项】
- 音效需求 【选项】

## 参考素材
- 图片参考（≤9张）【多行文本】 提示内容：“例如：@图片1作为首帧，@图片2为角色形象”
- 视频参考（≤3个）【多行文本】 提示内容：“例如：@视频1作为参考运镜”
- 音频参考（≤3个）【多行文本】 提示内容：“例如：@音频1作为整体配乐”

## 高亮提示语
！最多12个文件，视频/音频总市场≤15秒


## 自定义提示词
- 补充提示词 【多行文本】
