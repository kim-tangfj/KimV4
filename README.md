## 📖 简介

**Kim 多级分镜提示词助手** 是一款专为视频创作者和 AI 内容生成师设计的桌面应用，提供专业级的分镜管理和 AI 提示词生成功能。

通过结构化的分镜管理（项目 → 片段 → 镜头 三层结构），结合多个主流 AI 大模型（DeepSeek、豆包、通义千问等），帮助您快速生成高质量的视频生成提示词，大幅提升内容创作效率。

### ✨ 核心特性

- 🗂️ **多级分镜管理** - 项目/片段/镜头三层结构，清晰组织创作思路
- 🖼️ **分镜图片管理** - 支持上传、拖放、预览分镜参考图
- 🤖 **AI 提示词生成** - 集成多个主流大模型，自动生成专业提示词
- 📚 **素材库管理** - 项目素材库 + 片段素材库，分类管理参考资源
- 🎨 **双主题支持** - 浅色/深色主题，适配不同使用环境
- 💾 **自动保存** - 实时保存项目数据，防止意外丢失
- 🔌 **灵活配置** - 支持自定义 API、模板、选项

## 🚀 快速入门

### 第一步：创建项目

1. 启动应用后，点击左侧面板的 **+** 按钮
2. 输入项目名称、描述，选择画幅比例（如 16:9）
3. 点击「创建」按钮

### 第二步：添加片段

1. 选中刚创建的项目
2. 点击「片段」面板的 **+** 按钮
3. 输入片段名称（如「开场镜头」）
4. 设置片段时长、风格、情绪等属性

### 第三步：添加镜头

1. 选中刚创建的片段
2. 点击「镜头」面板的 **+** 按钮
3. 输入镜头名称（如「全景展示」）
4. 设置景别、镜头角度、运镜方式等

### 第四步：上传分镜图片（可选）

1. 在镜头属性面板，找到「分镜图片」区域
2. 点击上传区域选择图片，或从素材库拖放图片
3. 分镜图将显示在镜头列表卡片上

### 第五步：生成提示词

1. 选中一个镜头
2. 应用自动在「提示词」面板生成 AI 提示词
3. 点击 **📋 复制** 按钮即可使用

## 🔧 功能详解

### 1. 项目管理

- **新建项目**: 点击 + 按钮，填写项目信息
- **打开项目**: 点击项目卡片即可打开
- **删除项目**: 选中项目后点击 × 按钮（带确认）
- **项目属性**: 名称、描述、画幅比例、状态

### 2. 片段管理

- **新建片段**: 选中项目后点击 + 按钮
- **片段属性**: 名称、描述、时长、风格、情绪氛围、角色、场景设定
- **参考素材**: 图片/视频/音频引用
- **自定义提示词**: 补充提示词内容

### 3. 镜头管理

- **新建镜头**: 选中片段后点击 + 按钮
- **镜头属性**: 名称、景别、镜头角度、运镜方式、时长、内容描述
- **分镜图片**: 上传参考图片，支持拖放
- **详细信息**: 对白、情绪、备注

### 4. 素材库管理

#### 项目素材库
- 存放整个项目共享的素材资源
- 支持图片、视频、音频上传
- 可拖放素材到分镜图片区域

#### 片段素材库
- 存放特定片段专属的素材资源
- 独立于项目素材库
- 可拖放素材到分镜图片区域

### 5. 提示词生成

- **自动生成**: 选中镜头后自动生成
- **多模型支持**: DeepSeek、豆包、通义千问、阿里百炼
- **一键复制**: 点击 📋 按钮快速复制
- **导出功能**: 导出为文本或 JSON 格式

### 6. 设置配置

- **主题切换**: 浅色/深色主题
- **API 配置**: 配置各大模型 API Key
- **存储路径**: 自定义项目存储位置
- **模板管理**: 自定义分镜模板
- **自定义选项**: 管理景别、角度、运镜等选项

## 🔑 API Key 配置指南

应用支持多个主流 AI 大模型，需要配置对应的 API Key 才能使用提示词生成功能。

### DeepSeek

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册/登录账号
3. 进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制 API Key 到应用设置

### 豆包（字节）

1. 访问 [字节火山引擎](https://www.volcengine.com/)
2. 注册/登录账号
3. 进入「豆包大模型」页面
4. 创建 API Key
5. 复制 API Key 到应用设置

### 通义千问（阿里）

1. 访问 [阿里云百炼](https://bailian.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 进入「API-KEY 管理」页面
4. 创建 API Key
5. 复制 API Key 到应用设置

## ❓ 常见问题

### Q: 应用启动后显示空白？
A: 请检查是否安装了最新版本的 Electron，或尝试以管理员身份运行。

### Q: 提示词生成失败？
A: 请检查网络连接，并确认 API Key 配置正确。可在设置中点击「测试连接」验证。

### Q: 项目文件丢失？
A: 应用会自动保存数据。如遇到异常关闭，可在项目存储目录查找 `project.json` 文件。

### Q: 如何备份项目？
A: 直接复制整个项目文件夹即可。项目包含完整的目录结构和所有素材文件。

### Q: 支持 Mac 或 Linux 吗？
A: 当前版本仅支持 Windows。Mac 和 Linux 版本正在开发中。
