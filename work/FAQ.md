## 1. 安装问题

### Q1: 下载安装包后无法运行？

**A**: 可能原因及解决方案：
1. **系统不兼容**：本应用仅支持 Windows 10/11 (64 位)
2. **缺少运行库**：安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
3. **被杀毒软件拦截**：添加信任或暂时关闭杀毒软件
4. **文件损坏**：重新下载安装包

### Q2: 安装时提示"文件损坏"？

**A**: 
1. 重新下载安装包
2. 确保下载完整后再运行
3. 检查磁盘空间是否充足
4. 尝试以管理员身份运行

### Q3: 源码运行需要安装什么？

**A**: 
```bash
# 1. 安装 Node.js (v18+)
# 从 https://nodejs.org/ 下载安装

# 2. 克隆项目
git clone https://github.com/yourusername/kim-storyboard-assistant.git

# 3. 安装依赖
npm install

# 4. 启动应用
npm start
```

## 2. 使用问题

### Q4: 应用启动后显示空白？

**A**: 
1. 检查是否安装了最新版本的 Electron
2. 尝试以管理员身份运行
3. 删除 `node_modules` 文件夹，重新运行 `npm install`
4. 检查显卡驱动是否最新

### Q5: 如何创建第一个项目？

**A**: 
1. 点击左侧「项目」面板的 **+** 按钮
2. 输入项目名称（如「我的第一个视频」）
3. 选择画幅比例（推荐 16:9）
4. 点击「创建」

### Q6: 如何添加片段和镜头？

**A**: 
```
添加片段：
1. 选中项目
2. 点击「片段」面板 + 按钮
3. 输入片段名称
4. 点击「确定」

添加镜头：
1. 选中片段
2. 点击「镜头」面板 + 按钮
3. 输入镜头名称
4. 点击「确定」
```

### Q7: 自动保存多久生效？

**A**: 所有属性修改会在失焦后 500ms 自动保存，无需手动点击保存。

### Q8: 如何切换主题？

**A**: 
1. 点击菜单栏「文件」→「设置」
2. 选择「外观」标签
3. 选择「浅色主题」或「深色主题」
4. 立即生效

### Q9: 面板宽度可以调整吗？

**A**: 可以。将鼠标移到面板右侧边缘，当光标变为 `↔` 时，拖动调整宽度。

## 3. API 配置问题

### Q10: 如何配置 API Key？

**A**: 
1. 点击「文件」→「设置」→「API 配置」
2. 选择 AI 提供商（DeepSeek/豆包/通义千问）
3. 粘贴 API Key
4. 点击「测试连接」验证
5. 点击「保存」

### Q11: 在哪里获取 API Key？

**A**: 详见 [API 配置指南](./API 配置指南.md)

### Q12: 测试连接失败怎么办？

**A**: 
1. **检查网络**：确保网络连接正常
2. **检查 API Key**：重新复制粘贴，注意不要有空格
3. **关闭代理**：暂时关闭网络代理
4. **检查余额**：登录平台控制台查看余额
5. **稍后重试**：API 服务可能暂时不可用

### Q13: API Key 安全吗？

**A**: 非常安全：
- API Key 仅保存在本地配置文件中
- 不会上传到任何服务器
- 不会分享给第三方
- 建议定期更换

### Q14: 免费额度用完了怎么办？

**A**: 
1. 充值账户
2. 更换其他提供商的 API Key
3. 等待下月免费额度重置（部分平台）

### Q15: 哪个 AI 模型最好用？

**A**: 各模型特点：
- **DeepSeek**: 性价比高，适合长文本生成
- **豆包**: 响应速度快，适合快速迭代
- **通义千问**: 中文理解能力强，适合复杂场景

建议都尝试，选择最适合你的。

## 4. 素材管理问题

### Q16: 如何上传素材？

**A**: 
1. 打开项目素材库或片段素材库
2. 点击「上传素材」按钮
3. 选择图片/视频/音频文件
4. 支持多选

### Q17: 支持哪些文件格式？

**A**: 
- **图片**: jpg, jpeg, png, gif, webp, bmp
- **视频**: mp4, webm, ogg, mov, avi
- **音频**: mp3, wav, ogg, aac, flac

### Q18: 素材有大小限制吗？

**A**: 
- 建议单个文件不超过 10MB
- 大文件可能影响上传速度和性能
- 视频/音频总时长建议≤15 秒

### Q19: 如何删除素材？

**A**: 
1. 打开素材库
2. 右键点击素材
3. 选择「删除」
4. 确认删除

### Q20: 素材库不显示素材？

**A**: 
1. 点击刷新按钮
2. 检查素材文件是否被移动
3. 重启应用
4. 检查文件权限

### Q21: 如何拖放素材到分镜图片？

**A**: 
1. 打开素材库
2. 找到需要的图片
3. 拖放到分镜图片上传区域
4. 释放鼠标

### Q22: 项目素材库和片段素材库有什么区别？

**A**: 
| 特性 | 项目素材库 | 片段素材库 |
|------|------------|------------|
| 作用范围 | 整个项目共享 | 仅属于特定片段 |
| 存储位置 | `assets/images/` | `assets/shots/{shotId}/images/` |
| 适用场景 | 通用素材、角色设定 | 片段专属场景、道具 |

## 5. 提示词生成问题

### Q23: 提示词如何生成？

**A**: 
1. 选中一个镜头
2. 应用自动根据镜头属性生成提示词
3. 提示词显示在「提示词」面板

### Q24: 提示词可以自定义吗？

**A**: 可以：
1. 在镜头属性或片段属性中找到「补充提示词」
2. 输入额外要求
3. 会自动追加到生成的提示词中

### Q25: 如何复制提示词？

**A**: 
1. 点击提示词面板的 **📋** 按钮
2. 提示词自动复制到剪贴板
3. 可粘贴到 AI 视频生成工具

### Q26: 如何导出提示词？

**A**: 
1. 点击提示词面板的 **↓** 按钮
2. 选择导出格式（TXT/JSON）
3. 选择保存位置

### Q27: 提示词生成失败？

**A**: 
1. 检查 API Key 是否配置
2. 检查网络连接
3. 点击「测试连接」验证
4. 检查 API 余额

### Q28: 提示词内容不满意？

**A**: 
1. 修改镜头属性（内容描述、景别、角度等）
2. 添加补充提示词
3. 更换 AI 模型
4. 调整模板内容

## 6. 数据安全问题

### Q29: 项目数据保存在哪里？

**A**: 
默认保存位置：`文档/KimStoryboard/项目名称_时间戳/`

可在设置中更改存储路径。

### Q30: 如何备份项目？

**A**: 
直接复制整个项目文件夹即可。项目包含：
- `project.json` - 项目配置和分镜数据
- `assets/` - 所有素材文件

### Q31: 数据会同步到云端吗？

**A**: 
v1.0 版本所有数据都保存在本地，不会同步到云端。

云端同步功能计划在 v2.0 版本实现。

### Q32: 误删了项目如何恢复？

**A**: 
1. 检查回收站是否有删除的文件
2. 从备份中恢复
3. 使用数据恢复软件（如 Recuva）

### Q33: 应用崩溃了数据会丢失吗？

**A**: 
应用有自动保存机制：
- 属性修改后 500ms 自动保存
- 崩溃后可从 `project.json` 恢复
- 建议定期手动备份

### Q34: 可以导出项目分享给他人吗？

**A**: 
v1.0 版本暂不支持项目打包导出。

手动分享方法：
1. 复制整个项目文件夹
2. 压缩为 ZIP 文件
3. 发送给他人
4. 他人解压后放在 `文档/KimStoryboard/` 目录

项目导入导出功能计划在 v1.1 版本实现。
