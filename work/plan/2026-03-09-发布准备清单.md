### 四、打包配置 (P0 - 必须完成)

#### 4.1 Electron Builder 配置
- [ ] 安装 electron-builder
```bash
npm install --save-dev electron-builder
```

- [ ] 配置 package.json
```json
{
  "build": {
    "appId": "com.kimv4.storyboard",
    "productName": "Kim 分镜助手",
    "directories": {
      "output": "dist"
    },
    "files": [
      "src/**/*",
      "assets/**/*",
      "index.html",
      "styles.css",
      "package.json"
    ],
    "win": {
      "target": ["nsis", "portable"],
      "icon": "assets/icon.png"
    }
  }
}
```

#### 4.2 添加打包脚本
```json
{
  "scripts": {
    "pack": "electron-builder --dir",
    "dist": "electron-builder",
    "dist:win": "electron-builder --win"
  }
}
```

#### 4.3 应用图标
- [ ] 准备 256x256 PNG 图标
- [ ] 配置到 electron-builder
