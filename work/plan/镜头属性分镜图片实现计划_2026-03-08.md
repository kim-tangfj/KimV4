## 一、需求分析

### 1.1 功能需求

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 1. 点击上传 | 点击上传区域选择图片文件 | P0 |
| 2. 拖放上传 | 拖放本地图片文件到上传区域 | P0 |
| 3. 项目素材库拖放 | 从项目素材库拖放图片到分镜图片 | P0 |
| 4. 片段素材库拖放 | 从片段素材库拖放图片到分镜图片 | P0 |
| 5. 片段素材拖放限制 | 带📋标识的片段素材不可拖放 | P0 |
| 6. 独立存储 | 上传的图片保存到片段素材库 | P0 |

### 1.2 拖放规则

| 拖放源 | 目标 | 是否允许 | 说明 |
|--------|------|----------|------|
| 本地文件 | 分镜图片区域 | ✅ | 保存到片段素材库 |
| 项目素材库（无标识） | 分镜图片区域 | ✅ | 复制文件到片段素材库 |
| 项目素材库（📋标识） | 分镜图片区域 | ❌ | 提示"片段素材不可用于分镜" |
| 片段素材库（任意） | 分镜图片区域 | ✅ | 直接引用或复制 |

### 5.2 属性面板调用

```javascript
// propertyPanel.js
async function handleStoryboardImageUpload(file, shotId, sceneId) {
  const result = await window.electronAPI.uploadStoryboardImage({
    projectDir: currentProject.projectDir,
    filePath: file.path,
    shotId: shotId,
    sceneId: sceneId,
    fileName: file.name
  });
  
  if (result.success) {
    // 更新镜头数据
    updateSceneStoryboardImage(sceneId, result.asset);
    // 刷新属性面板
    renderPropertyPanel();
  }
}
```

## 七、风险评估

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 拖放事件冲突 | 拖放功能可能与其他拖放冲突 | 使用 dataTransfer 区分来源 |
| 文件命名冲突 | 同名文件覆盖 | 自动添加时间戳 |
| 素材库状态同步 | 删除分镜图后素材库需同步 | 添加引用计数或删除确认 |
| 大文件上传 | 上传慢、内存占用高 | 添加文件大小限制提示 |

## 九、注意事项

### 9.1 数据兼容

- 现有镜头数据不包含 `storyboardImage` 字段，需要向后兼容
- 旧项目可以正常打开和编辑

### 9.2 用户体验

- 拖放应有视觉反馈（高亮/禁止标识）
- 上传进度应有提示
- 预览图应支持点击放大

### 9.3 性能优化

- 图片预览应使用缩略图
- 大文件应限制大小（如 10MB）
- 上传应支持取消
