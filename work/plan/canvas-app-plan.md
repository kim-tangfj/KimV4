## 一、需求分析

### 1.1 产品定位

类似 **可灵 AI 画布** / **即梦画布** 的创作工具，集成：
- 📁 素材管理（图片/视频/音频）
- 🎨 画布编辑（拖放/缩放/图层）
- 🤖 AI 创作（文生图、图生图、图像编辑）
- 💾 作品保存与导出

### 1.2 核心功能

| 功能模块 | 子功能 | 优先级 |
|----------|--------|--------|
| **画布系统** | 画布创建/缩放/拖拽/网格 | P0 |
| **素材管理** | 图片上传/下载/拖放 | P0 |
| **文生图** | 提示词输入→生成图片→添加到画布 | P0 |
| **图生图** | 参考图片 + 提示词→生成新图片 | P1 |
| **图像编辑** | 局部重绘/扩图/去水印/风格迁移 | P1 |
| **图层管理** | 图层顺序/锁定/隐藏/删除 | P1 |
| **作品导出** | 导出为 PNG/JPG/项目文件 | P1 |

## 二、阿里百炼生图模型推荐

### 2.1 文生图模型

| 模型 | 价格 | 推荐理由 |
|------|------|----------|
| `wan2.6-t2i` | 0.20 元/张 | 万相 2.6 最新版，支持自由尺寸，性价比高 |
| `qwen-image-2.0-pro` | 0.50 元/张 | 高质量场景，复杂指令理解 |
| `z-image-turbo` | 0.10-0.20 元/张 | 轻量快速，适合预览 |

### 2.2 图生图模型

| 模型 | 价格 | 功能 |
|------|------|------|
| `wan2.5-i2i-preview` | 0.20 元/张 | 基于主体一致性的图像编辑 |
| `wanx-v1` | 0.16 元/张 | 风格迁移 |

### 2.3 图像编辑模型

| 模型 | 价格 | 功能 |
|------|------|------|
| `wan2.6-image` | 0.20 元/张 | 通用图像编辑 |
| `wanx-x-painting` | 免费 (500 张) | 局部重绘 |
| `image-out-painting` | 0.18 元/张 | 扩图 |
| `wanx2.1-imageedit` | 0.14 元/张 | 去水印/修复 |

## 七、API 调用示例

### 7.1 文生图

```javascript
async function textToImage(prompt, options) {
  const response = await window.electronAPI.callAIModel({
    model: 'wan2.6-t2i',
    input: { prompt: prompt },
    parameters: {
      size: options.size || '1024x1024',
      n: options.count || 1
    }
  });
  return response.images;
}
```

### 7.2 图生图

```javascript
async function imageToImage(referenceImage, prompt, options) {
  const response = await window.electronAPI.callAIModel({
    model: 'wan2.5-i2i-preview',
    input: {
      image: referenceImage,
      prompt: prompt
    },
    parameters: {
      strength: options.strength || 0.7
    }
  });
  return response.images;
}
```
