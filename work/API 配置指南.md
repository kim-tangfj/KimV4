## 1. DeepSeek API 配置

### 1.1 获取 API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 点击右上角「登录/注册」
3. 使用手机号或邮箱注册账号
4. 登录后进入「控制台」
5. 在左侧菜单选择「API Keys」
6. 点击「创建 API Key」按钮
7. 输入 API Key 名称（如：Kim 分镜助手）
8. 点击「确定」
9. **立即复制 API Key**（只显示一次，请妥善保存）

### 1.2 在应用中配置

1. 打开 Kim 分镜助手
2. 点击菜单栏「文件」→「设置」
3. 选择「API 配置」标签
4. 提供商选择「DeepSeek」
5. 粘贴 API Key 到输入框
6. 模型名称保持默认 `deepseek-chat`
7. 点击「测试连接」验证
8. 点击「保存」

### 1.3 计费说明

- **免费额度**：新用户注册赠送额度（具体以官网为准）
- **计费方式**：按 Token 数量计费
- **价格查询**：[DeepSeek 定价页面](https://platform.deepseek.com/pricing)

## 2. 豆包 API 配置

### 2.1 获取 API Key

1. 访问 [字节火山引擎](https://www.volcengine.com/)
2. 点击右上角「登录」
3. 使用手机号登录/注册
4. 登录后进入「控制台」
5. 在搜索框搜索「豆包大模型」
6. 进入豆包大模型控制台
7. 点击左侧「API Key 管理」
8. 点击「创建 API Key」
9. 输入名称，点击「确定」
10. **复制 API Key**（只显示一次）

### 2.2 在应用中配置

1. 打开 Kim 分镜助手
2. 点击「文件」→「设置」→「API 配置」
3. 提供商选择「豆包（字节）」
4. 粘贴 API Key
5. 模型名称保持默认 `doubao-pro-4k`
6. 点击「测试连接」
7. 点击「保存」

### 2.3 计费说明

- **免费额度**：新用户有一定免费额度
- **计费方式**：按 Token 计费
- **价格查询**：[火山引擎定价](https://www.volcengine.com/docs/6458/109944)

## 5. 测试 API 连接

配置完成后，建议测试连接是否正常：

### 测试步骤

1. 打开设置 → API 配置
2. 确认已选择提供商并填写 API Key
3. 点击「测试连接」按钮
4. 等待测试结果

### 测试结果

✅ **连接成功**
```
✓ 连接成功！
模型：deepseek-chat
余额：¥10.00（示例）
```

❌ **连接失败**
```
✗ 连接失败
错误信息：无效的 API Key
请检查 API Key 是否正确
```

### 常见错误

| 错误信息 | 原因 | 解决方案 |
|----------|------|----------|
| 无效的 API Key | API Key 填写错误 | 重新复制粘贴，注意不要有空格 |
| 余额不足 | 账户额度不足 | 充值或更换 API Key |
| 网络错误 | 网络连接问题 | 检查网络连接，关闭代理 |
| 模型不存在 | 模型名称错误 | 使用默认模型名称 |

## 6. 常见问题

### Q1: API Key 安全吗？

**A**: API Key 仅保存在本地配置文件中，不会上传到任何服务器。请妥善保管，不要分享给他人。

### Q2: 可以配置多个 API Key 吗？

**A**: v1.0 版本每个提供商只能配置一个 API Key。如需切换，请修改配置后保存。

### Q3: API Key 有有效期吗？

**A**: 大多数 API Key 长期有效，除非手动删除或账户被封禁。

### Q4: 免费额度用完了怎么办？

**A**: 需要充值或更换其他提供商的 API Key。

### Q5: 哪个模型最好用？

**A**: 不同模型各有特点：
- **DeepSeek**: 性价比高，适合长文本
- **豆包**: 响应速度快
- **通义千问**: 中文理解能力强

建议都尝试一下，选择最适合你的。

### Q6: 测试连接失败但 API Key 是对的？

**A**: 可能原因：
1. 网络连接问题 → 检查网络
2. 防火墙拦截 → 关闭防火墙或添加例外
3. 代理设置 → 关闭代理
4. API 服务暂时不可用 → 稍后重试

### Q7: 如何查看 API 使用量？

**A**: 登录各平台控制台，在「用量统计」或「账单」页面查看。

### Q8: API Key 泄露了怎么办？

**A**: 
1. 立即在原平台删除该 API Key
2. 重新创建新的 API Key
3. 在应用中更新配置
