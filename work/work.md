### 技术要点

#### 数据结构
```json
// 内置选项 (assets/default/options.json)
[
  {
    "group": "风格",
    "type": "写实风格",
    "style": "照片写实",
    "description": "高度还原现实光影...",
    "builtin": true
  }
]

// 自定义选项 (用户数据目录 options-custom.json)
{
  "version": "1.0",
  "customOptions": [
    {
      "id": "custom_1234567890",
      "group": "风格",
      "type": "自定义类型",
      "style": "自定义风格",
      "description": "描述...",
      "builtin": false
    }
  ]
}
```

#### 关键函数
- `showCustomOptionsModal()` - 打开管理弹窗
- `loadCustomOptionsList()` - 加载选项列表
- `loadGroupFilter()` - 加载组别筛选器
- `saveCustomOption()` - 保存自定义选项
- `deleteCustomOption()` - 删除自定义选项
