## 1. 快速入门

### 1.1 启动应用

- 从桌面快捷方式或开始菜单启动「Kim 分镜助手」
- 首次启动时，建议先配置 API Key（设置 → API 配置）

### 1.2 五步完成第一个项目

```
第 1 步：创建项目 → 点击「项目」面板 + 按钮
        ↓
第 2 步：添加片段 → 选中项目，点击「片段」面板 + 按钮
        ↓
第 3 步：添加镜头 → 选中片段，点击「镜头」面板 + 按钮
        ↓
第 4 步：上传分镜图 → 在镜头属性中上传参考图片
        ↓
第 5 步：生成提示词 → 选中镜头，自动 AI 生成提示词
```

### 1.3 操作流程图

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 创建项目  │ ──► │ 添加片段  │ ──► │ 添加镜头  │ ──► │ 生成提示词 │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
     │                │                │                │
     ▼                ▼                ▼                ▼
  设置画幅比例    设置时长风格    上传分镜图片    复制/导出使用
```

## 2. 界面布局

### 2.1 整体布局

应用采用四栏 + 底部面板布局：

```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│  ① 项目列表  │  ② 片段列表  │  ③ 镜头列表  │  ④ 提示词   │
│             │             │             │             │
│  [项目卡片]  │  [片段卡片]  │  [镜头卡片]  │  [提示词]   │
│  [项目卡片]  │  [片段卡片]  │  [镜头卡片]  │  [预览区]   │
│             │             │             │             │
│   [+] [↻]   │   [+] [×]   │ [+] [×] [□] │ [📋] [↓] [×]│
└─────────────┴─────────────┴─────────────┴─────────────┘
┌─────────────────────────────────────────────────────────┐
│  ⑤ 底部属性面板（可展开/收起）                          │
│     - 片段属性 / 镜头属性                               │
│     - 分镜图片上传 / 素材库管理                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 面板功能说明

| 面板 | 功能 | 主要操作 |
|------|------|----------|
| **① 项目列表** | 显示所有项目 | 新建、打开、删除、刷新 |
| **② 片段列表** | 显示当前项目的片段 | 新建、编辑、删除 |
| **③ 镜头列表** | 显示当前片段的镜头 | 新建、编辑、删除、切换视图 |
| **④ 提示词** | 显示 AI 生成的提示词 | 复制、导出、清空 |
| **⑤ 属性面板** | 编辑片段/镜头属性 | 表单填写、素材上传 |

### 2.3 面板宽度调整

将鼠标移到面板右侧边缘，当光标变为 `↔` 时，拖动调整宽度。

## 3. 项目管理

### 3.1 新建项目

1. 点击「项目」面板的 **+** 按钮
2. 填写项目信息：
   - **项目名称**（必填）：如「小刺猬团团登山梦」
   - **项目描述**（可选）：简要描述项目内容
   - **画幅比例**：选择 16:9、9:16、1:1 等
   - **状态**：草稿/制作中/已完成
3. 点击「创建」按钮

### 3.2 打开项目

- 在项目列表中点击任意项目卡片即可打开
- 打开后，「片段」面板会加载该项目的片段列表

### 3.3 删除项目

1. 选中要删除的项目
2. 点击项目卡片右上角的 **×** 按钮
3. 确认删除（⚠️ 删除后无法恢复）

### 3.4 项目属性编辑

打开项目后，在底部属性面板可编辑：
- 项目名称
- 项目描述
- 画幅比例
- 参考素材（图片/视频/音频）

### 3.5 项目数据存储

项目数据保存在：
```
文档/KimStoryboard/项目名称_时间戳/project.json
```

建议定期备份整个项目文件夹。

## 4. 片段管理

### 4.1 新建片段

1. 选中一个项目
2. 点击「片段」面板的 **+** 按钮
3. 输入片段名称（如「开场镜头」、「高潮部分」）
4. 点击「确定」

### 4.2 编辑片段属性

选中片段后，底部属性面板显示片段属性表单：

#### 基本信息
- **片段名称**：如「森林场景」
- **片段描述**：详细描述片段内容
- **视频时长**：单个片段最长 15 秒

#### 角色与场景
- **角色**：描述出场角色信息
- **场景设定**：如「室内演播室」、「户外森林」

#### 参考素材
- **图片参考**：最多 9 张，格式如 `@图片 1 作为首帧`
- **视频参考**：最多 3 个，格式如 `@视频 1 作为参考运镜`
- **音频参考**：最多 3 个，格式如 `@音频 1 作为整体配乐`

> ⚠️ **注意**：最多 12 个文件，视频/音频总时长≤15 秒

#### 风格设定
- **风格**：选择预设风格（如写实、卡通、水彩等）
- **情绪氛围**：选择情绪（如温馨、紧张、欢快等）

#### 视频参数
- **画幅比例**：16:9、9:16、1:1 等
- **视频时长**：单位秒

#### 声音设计
- **配乐风格**：选择背景音乐风格
- **音效需求**：选择音效类型

#### 自定义提示词
- **补充提示词**：输入额外的提示词或要求

### 4.3 删除片段

1. 选中要删除的片段
2. 点击「片段」面板的 **×** 按钮
3. 确认删除

### 4.4 片段自动保存

所有属性修改会在失焦后自动保存，无需手动点击保存按钮。

## 5. 镜头管理

### 5.1 新建镜头

1. 选中一个片段
2. 点击「镜头」面板的 **+** 按钮
3. 输入镜头名称（如「全景展示」、「特写镜头」）
4. 点击「确定」

### 5.2 编辑镜头属性

选中镜头后，底部属性面板显示镜头属性表单：

#### 基本信息
- **镜头名称**：如「森林中的团团」

#### 镜头信息
- **景别**：选择景别（如远景、全景、中景、近景、特写）
- **镜头角度**：选择角度（如平视、俯视、仰视）
- **运镜方式**：选择运镜（如固定镜头、推镜头、拉镜头、摇镜头）

#### 详细信息
- **时长**：单位秒
- **分镜图片**：上传参考图片（详见第 6 章）
- **内容描述**：详细描述镜头画面内容
- **对白内容**：如有台词或旁白
- **情绪描述**：如紧张、温馨、悲伤
- **其他备注**：其他需要说明的信息

### 5.3 镜头列表缩略图

上传分镜图片后，镜头卡片右侧会显示 50x50 的缩略图预览。

### 5.4 删除镜头

1. 选中要删除的镜头
2. 点击「镜头」面板的 **×** 按钮
3. 确认删除

### 5.5 镜头选中状态

- 选中的镜头会高亮显示
- 切换镜头时，选中状态会自动保持
- 提示词面板会实时更新为当前选中镜头的内容

## 6. 分镜图片

### 6.1 上传分镜图片

有三种方式上传分镜图片：

#### 方式一：点击上传
1. 在镜头属性面板，找到「分镜图片」区域
2. 点击上传区域
3. 选择图片文件（支持 jpg、png、webp、gif）
4. 点击「打开」

#### 方式二：本地文件拖放
1. 从电脑文件夹中拖放图片文件
2. 拖到分镜图片上传区域
3. 释放鼠标完成上传

#### 方式三：从素材库拖放
1. 打开项目素材库或片段素材库
2. 找到需要的图片
3. 拖放到分镜图片上传区域
4. 释放鼠标

### 6.2 分镜图片存储

上传的图片会自动保存到：
```
项目目录/assets/shots/{shotId}/images/
```

> 💡 **说明**：从项目素材库拖放时，会自动复制文件到片段素材库；从片段素材库拖放时，直接使用原文件。

### 6.3 预览分镜图片

上传成功后，上传区域会显示：
- 图片缩略图（100x100）
- 文件名称
- 删除按钮（红色 ×）

### 6.4 删除分镜图片

1. 点击分镜图片预览区域的红色 **×** 按钮
2. 确认删除

> ⚠️ **注意**：删除分镜图片只清空镜头的引用，不会删除物理文件。文件仍保留在片段素材库中，可重复使用。

### 6.5 镜头列表缩略图

上传分镜图片后，镜头列表中对应卡片右侧会显示缩略图：
- 尺寸：50x50 像素
- 位置：卡片右侧居中
- 效果：自适应填充，保持比例

## 7. 素材库管理

### 7.1 项目素材库

#### 打开素材库
1. 点击底部属性面板的「素材库」按钮
2. 或使用快捷键（待实现）

#### 上传素材
1. 点击「上传素材」按钮
2. 选择图片/视频/音频文件
3. 支持多选

#### 素材分类显示
- 🖼️ **图片**：jpg、jpeg、png、gif、webp、bmp
- 🎬 **视频**：mp4、webm、ogg、mov、avi
- 🎵 **音频**：mp3、wav、ogg、aac、flac

#### 素材操作
- **预览**：点击图片可放大预览
- **删除**：右键点击素材，选择删除
- **拖放**：拖放到分镜图片区域

### 7.2 片段素材库

#### 访问片段素材库
- 选中片段后，底部属性面板显示片段素材库
- 或使用快捷键（待实现）

#### 上传素材
1. 点击「上传素材」按钮
2. 选择文件上传
3. 素材仅属于当前片段

#### 素材操作
- **预览**：点击图片可放大预览
- **删除**：右键点击素材，选择删除
- **拖放**：拖放到分镜图片区域

### 7.3 素材库区别

| 特性 | 项目素材库 | 片段素材库 |
|------|------------|------------|
| **作用范围** | 整个项目共享 | 仅属于特定片段 |
| **存储位置** | `assets/images/` | `assets/shots/{shotId}/images/` |
| **适用场景** | 通用素材、角色设定 | 片段专属场景、道具 |

## 8. 提示词生成

### 8.1 自动生成

1. 选中一个镜头
2. 应用自动根据镜头属性生成提示词
3. 提示词显示在「提示词」面板

### 8.2 提示词内容

生成的提示词包含：
- 镜头基本信息（名称、时长）
- 画面描述（内容、景别、角度、运镜）
- 风格设定（风格、情绪）
- 参考素材（如有）
- 补充提示词（如有）

### 8.3 复制提示词

1. 点击提示词面板的 **📋** 按钮
2. 提示词自动复制到剪贴板
3. 可粘贴到 AI 视频生成工具

### 8.4 导出提示词

1. 点击提示词面板的 **↓** 按钮
2. 选择导出格式（TXT/JSON）
3. 选择保存位置

### 8.5 清空提示词

1. 点击提示词面板的 **×** 按钮
2. 清空当前提示词内容

### 8.6 自定义提示词

在镜头属性或片段属性的「补充提示词」字段输入额外要求，会追加到生成的提示词中。

## 9. 设置配置

### 9.1 打开设置

- 点击菜单栏「文件」→「设置」
- 或使用快捷键 `Ctrl + ,`

### 9.2 主题切换

1. 进入设置 → 外观
2. 选择「浅色主题」或「深色主题」
3. 立即生效

### 9.3 API 配置

1. 进入设置 → API 配置
2. 选择 AI 提供商（DeepSeek/豆包/通义千问/阿里百炼）
3. 输入 API Key
4. 输入模型名称（可选）
5. 点击「测试连接」验证
6. 点击「保存」

### 9.4 存储路径

1. 进入设置 → 存储
2. 点击「更改」按钮
3. 选择新的存储目录
4. 点击「保存」

默认存储路径：`文档/KimStoryboard/`

### 9.5 模板管理

1. 进入设置 → 模板管理
2. 查看默认模板
3. 可自定义模板内容
4. 支持备份和恢复模板

### 9.6 自定义选项

1. 进入设置 → 自定义选项
2. 管理景别、镜头角度、运镜、情绪氛围等选项
3. 可添加、编辑、删除自定义选项

## 10. 常见问题

### Q1: 应用启动后显示空白？

**解决方案**：
1. 检查是否安装了最新版本的 Electron
2. 尝试以管理员身份运行
3. 删除 `node_modules` 文件夹，重新运行 `npm install`

### Q2: 提示词生成失败？

**解决方案**：
1. 检查网络连接是否正常
2. 确认 API Key 配置正确
3. 在设置中点击「测试连接」验证
4. 检查 API 额度是否充足

### Q3: 项目文件丢失？

**解决方案**：
1. 应用会自动保存数据
2. 在项目存储目录查找 `project.json` 文件
3. 检查是否有备份文件

### Q4: 如何备份项目？

**解决方案**：
直接复制整个项目文件夹即可。项目包含完整的目录结构和所有素材文件。

### Q5: 分镜图片上传失败？

**解决方案**：
1. 检查图片格式是否支持（jpg/png/webp/gif）
2. 检查文件大小（建议不超过 10MB）
3. 检查磁盘空间是否充足

### Q6: 素材库不显示素材？

**解决方案**：
1. 点击素材库刷新按钮
2. 检查素材文件是否被移动或删除
3. 重启应用

### Q7: 支持 Mac 或 Linux 吗？

**答案**：当前版本仅支持 Windows。Mac 和 Linux 版本正在开发中。

### Q8: 如何获取 API Key？

**答案**：详见 [API 配置指南](./API 配置指南.md)

### Q9: 自动保存的频率是多久？

**答案**：所有属性修改会在失焦后 500ms 自动保存。

### Q10: 可以撤销误操作吗？

**答案**：v1.0 版本暂不支持撤销功能，请谨慎操作。该功能计划在后续版本中添加。
