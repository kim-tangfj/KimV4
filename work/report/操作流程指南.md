## 1. 快速入门

### 1.1 启动应用

双击应用图标或在项目目录运行：

```bash
npm start
```

### 1.2 界面布局

应用采用**四栏 + 底部双面板**布局：

```
┌────────────┬────────────┬────────────┬────────────┐
│   项目     │   片段     │   镜头     │  提示词    │
│   列表     │   列表     │   列表     │  预览      │
│            │            │            │            │
└────────────┴────────────┴────────────┴────────────┘
┌───────────────────────────────────────────────────┐
│   属性面板（展开/收起）                           │
├───────────────────────────────────────────────────┤
│   片段素材库（展开/收起）                         │
└───────────────────────────────────────────────────┘
```

| 面板 | 功能 |
|------|------|
| **项目列表** | 显示所有项目，支持新建/打开/删除 |
| **片段列表** | 显示当前项目的片段，支持新建/删除 |
| **镜头列表** | 显示当前片段的镜头，支持新建/删除/视图切换 |
| **提示词预览** | 显示自动生成的提示词，支持复制/导出 |
| **属性面板** | 编辑选中片段/镜头的属性 |
| **片段素材库** | 管理当前片段的专属素材 |

### 1.3 基本工作流程

```
① 创建项目 → ② 创建片段 → ③ 创建镜头 → ④ 编辑属性 → ⑤ 生成提示词
```

## 2. 项目创建

### 2.1 手动创建项目

**适用场景**: 已有完整的分镜脚本数据，或想手动输入项目结构

**操作步骤**:

1. 点击左上角 **"+"** 按钮（新建项目）
2. 在弹窗中选择 **"手动创建"** 标签
3. 填写项目信息：
   - **项目名称** *（必填）*
   - **项目描述**（可选）
   - **默认画幅**: 16:9 / 9:16 / 1:1 / 4:3 / 3:4
   - **项目剧本** *（必填）*: 输入产品/视频基本信息
   - **AI 提示词** *（必填）*: 粘贴 AI 生成的结构化 JSON 数据

4. 点击 **"创建项目"**

**剧本内容示例**:

```
产品名称：智能运动手表
产品卖点：心率监测、GPS 定位、50 米防水、续航 7 天
视频类型：商品介绍
目标受众：运动爱好者、健身人群
```

**AI 提示词格式示例** (project.json):

```json
{
  "project": {
    "name": "智能手表宣传片",
    "description": "展示产品核心功能",
    "aspectRatio": "16:9",
    "status": "draft"
  },
  "shots": [
    {
      "name": "开场展示",
      "description": "产品外观展示",
      "duration": 10,
      "scenes": [
        {
          "name": "手表特写",
          "shotType": "特写",
          "angle": "平拍",
          "camera": "推镜头",
          "content": "手表在黑色背景下缓缓旋转",
          "duration": 3,
          "dialogue": "",
          "emotion": "科技感"
        }
      ]
    }
  ]
}
```

### 2.2 AI 生成项目

**适用场景**: 只有剧本描述，需要 AI 自动生成完整分镜结构

**操作步骤**:

1. 点击左上角 **"+"** 按钮
2. 在弹窗中选择 **"AI 生成"** 标签
3. 填写项目信息：
   - **项目名称** *（必填）*
   - **项目描述**（可选）
   - **项目剧本** *（必填）*: 详细描述产品/视频信息
   - **默认画幅**: 选择画幅比例
   - **选择 AI 提供商**: DeepSeek / 豆包 / 通义千问 / 阿里百炼

4. 点击 **"✨ 生成提示词"** 按钮
5. 等待 AI 返回结构化数据（预览在下方显示）
6. 确认数据无误后，点击 **"创建项目"**

**AI 生成提示词示例**:

```
你是一位专业的视频分镜脚本助手。请将以下剧本内容转换为结构化的 JSON 格式片段数据。

剧本内容：
产品名称：智能运动手表
产品卖点：心率监测、GPS 定位、50 米防水、续航 7 天
视频类型：商品介绍
目标受众：运动爱好者、健身人群

重要约束：
1. 每个片段 (shot) 的总时长不能超过 15 秒
2. 片段内所有镜头 (scenes) 的时长相加不能超过 15 秒
3. 每个镜头的时长建议在 2-5 秒之间
4. 镜头描述要细致明确，连贯不突兀，便于视觉化呈现
5. 必须包含景别、运镜、风格、情绪氛围信息
...
```

**注意事项**:
- 首次使用需先在设置中配置 API Key
- AI 生成可能需要 10-30 秒，请耐心等待
- 生成失败请检查 API Key 是否正确或网络连接

## 3. 分镜管理

### 3.1 片段管理

#### 3.1.1 新建片段

**前提**: 已选中项目

**操作步骤**:
1. 点击片段面板 **"+"** 按钮
2. 在属性面板填写片段信息：
   - **名称**: 片段名称（如"开场展示"）
   - **描述**: 片段内容描述
   - **时长**: 片段总时长（秒）
   - **画幅**: 16:9 / 9:16 / 1:1 等
   - **风格**: 从下拉列表选择（如"照片写实"）
   - **情绪氛围**: 从下拉列表选择（如"暖阳治愈"）

3. 按 **Enter** 或点击其他位置自动保存

#### 3.1.2 编辑片段属性

1. 在片段列表中**点击选中**片段
2. 展开底部**属性面板**（如已收起）
3. 修改属性值，自动保存

#### 3.1.3 删除片段

1. 在片段列表中**选中**片段
2. 点击片段面板 **"×"** 按钮
3. 在确认对话框中点击 **"确认"**

**注意**: 删除片段将同时删除其所有镜头和素材！

---

### 3.2 镜头管理

#### 3.2.1 新建镜头

**前提**: 已选中片段

**操作步骤**:
1. 点击镜头面板 **"+"** 按钮
2. 在属性面板填写镜头信息：
   - **名称**: 镜头描述（如"手表特写"）
   - **景别**: 大全景/远景/中景/近景/特写等
   - **角度**: 平拍/仰拍/俯拍/侧拍等
   - **运镜**: 推镜头/拉镜头/摇镜头/跟拍等
   - **内容**: 详细画面内容描述
   - **时长**: 镜头时长（秒）
   - **台词**: 角色对话（可选）
   - **情绪**: 情绪氛围（可选）
   - **备注**: 其他说明（可选）

3. 按 **Enter** 或点击其他位置自动保存

#### 3.2.2 编辑镜头属性

1. 在镜头列表中**点击选中**镜头
2. 展开底部**属性面板**
3. 修改属性值，自动保存

#### 3.2.3 删除镜头

1. 在镜头列表中**选中**镜头
2. 点击镜头面板 **"×"** 按钮
3. 在确认对话框中点击 **"确认"**

#### 3.2.4 切换视图模式

点击镜头面板 **"□"** 按钮，在**列表视图**和**卡片视图**之间切换。

---

### 3.3 属性编辑技巧

#### 3.3.1 使用下拉选择

点击字段右侧下拉箭头，从预设选项中选择：

| 字段 | 可选数量 |
|------|----------|
| **景别** | 10 种 |
| **角度** | 10 种 |
| **运镜** | 12 种 |
| **风格** | 50+ 种 |
| **情绪氛围** | 30+ 种 |

#### 3.3.2 自定义选项

如预设选项不满足需求，可添加自定义选项：

1. 点击菜单栏 **"工具"** → **"自定义选项管理"**
2. 点击右侧 **"+"** 按钮
3. 填写信息：
   - **组别**: 选择或输入新组别（如"风格"）
   - **类型**: 分类名称（如"写实风格"）
   - **风格名称**: 具体名称（如"照片写实"）
   - **描述**: 特点说明

4. 点击 **"保存"**

## 4. 素材管理

### 4.1 项目素材库

**用途**: 存储整个项目的共享素材资源

#### 4.1.1 打开项目素材库

1. 点击底部**项目素材库**面板标题栏展开
2. 或点击右侧 **"📁"** 按钮打开侧边窗体

#### 4.1.2 上传素材

**方法 1: 点击上传**
1. 点击上传区域
2. 在文件选择对话框中选择文件（可多选）
3. 自动上传并分类存储

**方法 2: 拖放上传**
1. 拖拽文件到上传区域
2. 释放鼠标自动上传

**支持格式**:
- **图片**: jpg, jpeg, png, gif, webp, bmp
- **视频**: mp4, webm, ogg, mov, avi
- **音频**: mp3, wav, ogg, aac, flac

#### 4.1.3 素材分类筛选

点击分类标签筛选素材：
- 📁 **全部素材**
- 📁 **图片**
- 📁 **视频**
- 📁 **音频**

#### 4.1.4 搜索素材

在搜索框输入关键词，实时过滤素材列表。

#### 4.1.5 预览素材

**方法 1: 点击缩略图**
- 点击任意素材缩略图
- 在侧边预览面板查看

**方法 2: 右键菜单**
- 右键点击素材
- 选择 **"👁️ 查看"**

#### 4.1.6 删除素材

**方法 1: 右键菜单**
1. 右键点击素材
2. 选择 **"🗑️ 删除"**
3. 确认删除

**方法 2: 预览删除**
1. 打开预览模态框
2. 点击 **"删除"** 按钮
3. 确认删除

**注意**: 
- 删除前会检查是否被镜头引用
- 如被引用，会显示警告提示
- 删除后无法恢复！

---

### 4.2 片段素材库

**用途**: 存储当前片段的专属素材资源

#### 4.2.1 打开片段素材库

1. 选中一个片段
2. 点击底部**片段素材库**面板标题栏展开

#### 4.2.2 上传素材

**方法 1: 点击上传**
1. 点击上传区域
2. 选择文件（可多选）
3. 自动上传并关联到当前片段

**方法 2: 拖放上传**
1. 拖拽文件到上传区域
2. 释放鼠标自动上传

**注意**: 
- 片段素材独立存储到 `assets/shots/{shotId}/` 目录
- 与项目素材库完全独立

#### 4.2.3 预览素材

1. 点击任意素材缩略图
2. 在预览模态框查看
3. 支持图片/视频/音频播放

#### 4.2.4 删除素材

**方法 1: 右键菜单**
1. 右键点击素材
2. 选择 **"删除"**
3. 确认删除

**方法 2: 预览删除**
1. 打开预览模态框
2. 点击 **"删除"** 按钮
3. 确认删除

## 5. 提示词生成

### 5.1 自动生成

**触发条件**: 选中任意镜头

**生成规则**:
- 自动组装镜头属性为格式化提示词
- 包含时长、景别、角度、运镜、内容、台词、情绪

**示例输出**:

```
## 镜头 1
**0-3 秒**：[特写、平拍、推镜头]，手表在黑色背景下缓缓旋转，展现产品精致外观（科技感）

## 镜头 2
**3-8 秒**：[中景、低角度仰拍、环绕跟拍]，运动员佩戴手表跑步，手表屏幕显示心率数据（激情澎湃）
【对白】旁白："实时心率监测，掌握每一次心跳"
```

---

### 5.2 复制提示词

1. 确保已选中镜头（提示词已生成）
2. 点击提示词面板 **"📋"** 按钮
3. 提示词复制到剪贴板

---

### 5.3 导出提示词

1. 点击提示词面板 **"↓"** 按钮
2. 选择保存位置
3. 输入文件名
4. 保存为 `.txt` 文件

---

### 5.4 清空提示词

1. 点击提示词面板 **"✕"** 按钮
2. 提示词区域清空

---

### 5.5 批量生成（待实现）

**计划功能**: 为整个片段/项目生成完整提示词

1. 选中片段
2. 点击菜单栏 **"生成"** → **"批量生成提示词"**
3. 选择输出格式（Markdown / TXT）
4. 保存文件

### 6.4 大模型 API 配置

#### 6.4.1 选择提供商

1. 在设置面板找到 **"大模型 API 设置"**
2. 从下拉列表选择提供商：
   - **DeepSeek**
   - **豆包 (字节)**
   - **通义千问 (阿里)**
   - **阿里百炼**

#### 6.4.2 配置 API Key

以 DeepSeek 为例：

1. 选择 **"DeepSeek"** 提供商
2. 输入 **API Key**
   - 点击 **"👁"** 按钮可显示/隐藏
3. 确认 **模型名称**（默认 `deepseek-chat`）
4. 点击 **"测试连接"** 验证配置
5. 显示 **"✅ 连接成功"** 后点击 **"保存"**

#### 6.4.3 获取 API Key

**DeepSeek**:
1. 访问 https://platform.deepseek.com/
2. 登录/注册账号
3. 进入 API 管理页面
4. 创建 API Key

**豆包 (字节)**:
1. 访问 https://www.volcengine.com/
2. 登录/注册账号
3. 进入方舟大模型平台
4. 创建 API Key

**通义千问 (阿里)**:
1. 访问 https://dashscope.console.aliyun.com/
2. 登录/注册阿里云账号
3. 创建 API Key

**阿里百炼**:
1. 访问 https://bailian.console.aliyun.com/
2. 登录/注册阿里云账号
3. 创建 API Key

#### 6.8.4 删除自定义选项

## 7. 常见问题

### 7.1 项目创建

**Q: AI 生成项目失败，提示"API Key 未配置"**

A: 请先在设置中配置 API Key：
1. 打开设置（`Ctrl + ,`）
2. 选择 AI 提供商
3. 输入 API Key
4. 测试连接
5. 保存

**Q: 手动创建项目时，AI 提示词格式错误**

A: 请确保粘贴的 JSON 数据格式正确：
- 使用 JSON 验证工具检查格式
- 确保包含 `project` 和 `shots` 字段
- 确保 `shots` 是数组

---

### 7.2 分镜管理

**Q: 片段/镜头删除后无法撤销**

A: 当前版本不支持撤销功能，删除前请确认。建议：
- 定期备份项目（复制项目文件夹）
- 删除前导出提示词备份

**Q: 如何批量修改片段/镜头属性**

A: 当前版本不支持批量修改，需逐个编辑。该功能已在计划中。

---

### 7.3 素材管理

**Q: 上传素材失败，提示"项目未加载"**

A: 请确保：
1. 已打开项目
2. 项目数据已正确加载
3. 重新打开项目后重试

**Q: 视频素材预览黑屏**

A: 可能原因：
- 视频编码格式不支持（推荐 H.264/MP4）
- 视频文件损坏
- 浏览器不支持该编码格式

解决方法：
- 使用转码工具转换为 MP4 格式
- 重新上传素材

**Q: 删除素材时提示"该素材正被镜头引用"**

A: 说明该素材被某个镜头的 `materials` 字段引用：
1. 找到引用该素材的镜头
2. 移除镜头的素材关联
3. 再次删除素材

---

### 7.4 提示词生成

**Q: 提示词生成不完整**

A: 请检查：
- 镜头属性是否填写完整
- 镜头数据是否保存成功
- 刷新页面后重试

**Q: 如何自定义提示词格式**

A: 当前版本提示词格式固定，自定义功能已在计划中。

---

### 7.5 设置配置

**Q: 切换主题后重启应用恢复原样**

A: 主题设置会自动保存，如未保存可能是：
- 应用异常退出
- 设置文件损坏

解决方法：
- 重新设置主题
- 检查设置文件路径是否有写入权限

**Q: API 测试连接失败**

A: 可能原因：
- API Key 错误
- 网络连接问题
- API 服务不可用

解决方法：
1. 检查 API Key 是否正确（无多余空格）
2. 检查网络连接
3. 访问 API 提供商官网确认服务状态
4. 尝试更换 API 提供商

---

### 7.6 其他问题

**Q: 应用卡顿或响应慢**

A: 可能原因：
- 项目数据过大
- 素材文件过多
- 系统资源不足

解决方法：
1. 关闭其他占用资源的程序
2. 清理不必要的素材文件
3. 拆分大项目为多个小项目

**Q: 数据丢失**

A: 预防措施：
- 定期备份项目文件夹
- 使用模板备份功能
- 开启自动保存

恢复方法：
- 从备份文件恢复
- 检查项目目录是否有 `project.json` 文件
