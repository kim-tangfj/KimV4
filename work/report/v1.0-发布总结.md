## 📋 v1.0 版本亮点

### 核心特性

1. **多级分镜管理** - 项目→片段→镜头三层结构
2. **分镜图片上传** - 支持本地上传和素材库拖放
3. **镜头缩略图** - 实时显示分镜图预览
4. **AI 提示词生成** - 集成多个主流大模型
5. **双素材库** - 项目素材库 + 片段素材库
6. **双主题支持** - 浅色/深色主题切换
7. **自动保存** - 实时保存，防止丢失
8. **灵活配置** - 自定义 API、模板、选项

### 技术优势

- Electron 40.6.1 框架
- 安全的 IPC 通信机制
- 完善的错误处理
- 高性能渲染优化
- 本地数据存储
