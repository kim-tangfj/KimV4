# Kim 多级分镜提示词助手 - 完整优化版项目开发方案

# 一、项目核心定位

**产品定位**：面向视频创作者、AI内容生成师的多级分镜管理与提示词自动化生成工具，通过结构化的项目-片段-镜头层级管理，降低分镜创作门槛，提升AI提示词生成的精准度和复用性。

**核心价值**：可视化横向层级管理（一览无余）+ 自动化提示词生成 + 本地化文件安全存储 + 底部显隐式属性操作，兼顾操作效率与易用性；同时采用黑白灰简约配色、线条图标及深浅双主题，保障界面简洁、适配性强。

# 二、功能架构（完整优化版）

## 2.1 整体架构逻辑

保留「项目管理 > 片段(Shot)管理 > 镜头(Scene)管理」的层级关系，优化提示词生成逻辑，结合横向平铺布局，形成完整闭环：全局项目配置 + 片段属性 + 镜头精细化参数 + 剧本/参考素材 → 分层级提示词生成（单镜头/整片段/全项目），所有核心内容横向平铺展示，属性操作集中于底部显隐面板。

## 2.2 页面交互设计（核心优化：横向平铺+底部显隐操作+视觉规范）

采用「主体四栏横向平铺 + 底部显隐属性操作面板」的结构，彻底摒弃选项卡切换和顶部全局操作栏，将操作按钮分散集成于各栏标题区域，实现项目>片段>镜头>提示词一览无余，联动逻辑直观清晰；同时严格遵循视觉规范，以黑白灰三色为主，搭配简约线条图标，支持深浅双主题切换。

### 2.2.1 视觉规范（新增核心优化）

#### （1）配色方案

整体采用黑白灰三色为主，无鲜艳颜色，保持界面简洁干净，兼顾可读性与简约质感，具体规范如下：

- 基础色：白色（背景底色）、浅灰色（区域分隔、次要元素）、深灰色（文本、核心线条）、黑色（重点文本、图标主色）；

- 交互色：中灰色（ hover 状态）、深灰色（选中状态边框/背景，无鲜艳高亮）、浅灰色（禁用状态）；

- 辅助色：仅使用极浅灰色（提示文本、分割线），无其他额外色彩，确保整体视觉统一。

#### （2）图标规则

全量使用简约线条风格图标，避免复杂设计，确保不同分辨率下清晰可辨，具体规范如下：

- 线条规格：统一采用1px粗细线条，无填充、无渐变、无阴影，仅用黑色/深灰色呈现；

- 图标尺寸：核心操作图标（新建、删除、保存等）统一为20×20px，次要图标（筛选、显隐等）统一为16×16px，适配不同屏幕；

- 设计原则：图标造型简洁，贴合功能含义（如新建用「+」、删除用「×」、保存用「√」），无多余装饰元素，确保辨识度。

#### （3）双主题设计

提供浅色和深色两种主题，用户可在系统设置中自由切换，所有元素在两种主题下均保证良好的可读性和视觉效果，具体适配如下：

- 浅色主题（默认）：白色背景，深灰色文本，浅灰色分隔线，黑色图标，中灰色交互状态；

- 深色主题：深灰色背景（非纯黑），浅灰色文本，中灰色分隔线，浅灰色图标，灰色交互状态；

- 适配要求：主题切换时，所有文本、图标、面板、分隔线颜色同步切换，无模糊、无色彩断层，提示词预览区语法高亮颜色同步适配对应主题（浅色用深灰标亮，深色用浅灰标亮）。

### 2.2.2 页面分区详细设计（无表格，纯文本呈现）

#### 1. 第一栏：项目列表

宽度占比15%，以卡片形式展示所有项目，卡片采用浅灰色边框，无填充色，包含内容：项目名称、项目状态（灰色系标签：草稿/进行中/已完成/已归档）、片段数量。

**标题栏操作**：
- 左侧显示「项目」标题（深灰色字体）；
- 右侧放置「项目菜单」按钮（简约线条图标：≡ 或 ⋮），点击展开下拉菜单，包含：
  - **新增项目**：弹出创建项目表单（输入名称、描述、默认画幅等）；
  - **修改当前项目**：选中项目后可用，弹出修改项目信息表单；
  - **删除当前项目**：选中项目后可用，点击后二次确认（弹窗为黑白灰样式）；
  - **打开资源文件管理器**：选中项目后可用，直接跳转到该项目`assets`文件夹所在路径。
- 菜单项悬停显示浅灰色背景，点击后执行对应操作。

**交互规则**：
- 点击项目卡片，卡片呈现深灰色边框+浅灰色背景（选中态，无鲜艳色），第二栏加载该项目下所有片段，第三、四栏同步联动更新；
- 双击项目卡片可快速编辑项目名称；
- 卡片悬停显示更多操作按钮（修改/归档），按钮图标为简约线条；
- 顶部设置浅灰色筛选栏，可按状态快速筛选项目。

#### 2. 第二栏：片段列表

宽度占比20%，展示选中项目下的所有片段，以列表项形式呈现，列表项采用浅灰色分隔线区分，包含内容：片段名称、时长/画幅、镜头数量、状态标签（灰色系：草稿/完成）。

**标题栏操作**：
- 左侧显示「片段」标题（深灰色字体）；
- 右侧放置「片段操作」按钮组或下拉菜单，包含：
  - **新建片段**：基于当前选中项目，弹出创建片段表单；
  - **删除片段**：选中一个或多个片段后可用，批量删除；
  - **批量操作**：如批量导出、批量排序等（可展开二级菜单）。
- 若无项目选中，新建片段按钮置灰；若无片段选中，删除按钮置灰。

**交互规则**：
- 点击片段项，项呈现浅灰色背景（选中态），第三栏加载该片段下所有镜头，第四栏同步生成对应片段的提示词；
- 双击片段项可快速编辑片段核心属性（名称/时长）；
- 支持拖拽调整片段顺序，调整后同步更新project.json；
- 无片段时显示浅灰色提示文本「暂无片段，点击+新建」，搭配简约「+」线条图标（点击直接触发新建片段）。

#### 3. 第三栏：镜头列表

宽度占比25%，展示选中片段下的所有镜头，支持卡片/列表视图切换（切换按钮为简约线条图标），包含内容：镜头编号/名称、镜头类型（特写/近景等）/时长、核心描述（缩略，hover显示完整内容）、启用/禁用开关（简约线条样式，黑白灰配色）。

**标题栏操作**：
- 左侧显示「镜头」标题（深灰色字体）；
- 右侧放置「镜头操作」按钮组或下拉菜单，包含：
  - **新建镜头**：基于当前选中片段，弹出创建镜头表单；
  - **删除镜头**：选中一个或多个镜头后可用，批量删除；
  - **批量启用/禁用**：批量切换镜头启用状态；
  - **视图切换**：卡片/列表视图切换按钮。
- 若无片段选中，新建镜头按钮置灰；若无镜头选中，删除/启用禁用按钮置灰。

**交互规则**：
- 点击镜头项，项呈现浅灰色背景（选中态），第四栏实时更新该镜头+片段+项目整合的提示词；
- 双击镜头项可快速编辑镜头核心属性（名称/时长/镜头类型）；
- 支持拖拽调整镜头顺序，调整后提示词时间轴同步重新排序；
- 启用/禁用开关实时影响提示词生成（禁用则提示词中剔除该镜头）；
- 无镜头时显示浅灰色提示文本「暂无镜头，点击+新建或从其他片段导入」，搭配简约「+」线条图标（点击直接触发新建镜头）。

#### 4. 第四栏：提示词预览

宽度占比40%，分区域展示，核心无切换，包含内容：顶部（提示词类型切换：单镜头/整片段/全项目，切换按钮为简约线条图标）、主体（提示词编辑预览区，支持手动编辑，语法高亮采用黑白灰渐变，无鲜艳色）、底部（快捷操作：复制/清空/模板套用/导出，搭配对应线条图标）。

**标题栏操作**：
- 左侧显示「提示词」标题（深灰色字体）；
- 右侧放置「提示词操作」按钮组，包含：
  - **复制**：复制当前预览区提示词到剪贴板；
  - **导出**：导出提示词为TXT/JSON文件；
  - **清空**：清空手动编辑内容（若为自动生成则恢复自动生成）；
  - **模板套用**：选择预设模板套用。
- 所有按钮悬停显示功能提示。

**交互规则**：
- 选中层级变化/编辑任意属性，提示词实时自动更新，修改位置显示2秒浅灰色「已更新」轻提示；
- 手动编辑提示词，可选择「覆盖自动生成」或「保存为自定义提示词」；
- 切换提示词类型，预览区同步展示对应范围的提示词，无页面跳转；
- 无选中内容时显示浅灰色提示文本「请先选中项目>片段>镜头，自动生成提示词」。

#### 5. 底部属性操作面板

宽度占比100%，高度可调整（默认40%/最小20%/最大60%），面板边框为浅灰色，背景贴合当前主题（浅色主题为白色，深色主题为深灰色），包含内容：顶部标签（基础信息/高级属性/资源管理，同一面板内切换，标签采用浅灰色分隔，选中标签为深灰色）、主体（表单式属性编辑区，分组布局，相关属性聚合，表单元素为黑白灰配色）、底部（保存/重置按钮，搭配简约线条图标）。

**标题栏操作**：
- 左侧显示「属性」标题（深灰色字体）；
- 右侧放置「面板显隐」按钮（简约线条图标：▼ 或 ▲），点击可收起/展开面板；面板收起时仅显示标题栏，点击标题栏也可展开。

**交互规则**：
- 显隐交互：展开（点击显隐按钮或标题栏展开），收起（点击显隐按钮或右上角收起图标），拖拽调整（顶部浅灰色拖拽条，可手动调整高度）；
- 表单适配规则为，选中项目显示项目属性（名称/描述/状态/默认画幅/存储路径等），选中片段显示片段属性（名称/时长/风格/情绪/参考素材等），选中镜头显示镜头属性（名称/类型/角度/内容/台词/分镜图等）；
- 校验逻辑为表单即时校验（如非数字时长实时显示浅灰色提示），保存前必填项校验，校验提示为浅灰色文本。

### 2.2.3 关键视觉与联动强化设计

#### （1）层级联动视觉强化

- 选中态统一：所有选中的项目/片段/镜头均采用「深灰色边框+浅灰色背景」的高亮样式，完全贴合黑白灰配色，视觉上形成清晰的选中链路，一眼识别当前操作层级；

- 数据联动反馈：任意层级属性修改后，相关区域实时刷新（如修改镜头时长，提示词预览区时间轴立即更新），修改位置显示2秒浅灰色「已更新」轻提示，无鲜艳色彩干扰；

- 分隔与阴影：四栏之间用浅灰色细分隔线划分，无多余阴影，仅通过线条区分区域，保持界面简洁；各栏滚动条优化为简约浅灰色样式，悬停时变为深灰色，提升浏览体验，贴合整体配色。

#### （2）响应式适配（补充优化）

适配不同屏幕尺寸，避免横向布局信息杂乱，同时保障黑白灰配色、线条图标在各分辨率下的适配性：

- 大屏（≥1920px）：四栏按15%/20%/25%/40%比例完整展示，所有内容无删减，线条图标清晰可见；

- 中屏（1440-1919px）：第三栏（镜头列表）隐藏次要信息（核心描述缩略为前20字），第四栏适当缩小字体，线条图标同步缩小适配，保持清晰；

- 小屏（<1440px）：支持「栏收缩」，点击栏右侧简约线条「←」按钮，可将对应栏收缩为仅显示标题，释放空间给核心栏（提示词预览栏优先保留完整显示），收缩按钮贴合黑白灰配色。

## 2.3 功能点精细化设计（完整优化，无表格）

### 2.3.1 项目管理（核心优化+补充）

#### 1. 项目创建

详细设计：输入项包含项目名称（必填）、项目描述（选填）、默认画幅（16:9/9:16等，下拉选择，下拉框为黑白灰配色，线条样式）；自动生成内容包括：项目文件夹（命名规则：项目名称_创建时间戳，避免重名）、assets子文件夹（分images/videos/audios子目录，便于资源分类）、project.json（含基础字段+默认测试数据）。

补充优化点：增加「创建前校验」，检测同路径下是否有同名项目，避免覆盖，校验提示为浅灰色文本；测试数据生成规则为每个新项目默认生成1个测试片段，每个测试片段默认生成3-5个随机属性的测试镜头，测试数据ID采用时间戳+随机数，避免重复。

#### 2. 项目修改

详细设计：可修改内容包括项目名称、描述、默认画幅、项目状态（草稿/进行中/已完成/已归档）；修改后自动更新project.json的project字段，同步更新页面显示，修改界面为黑白灰表单样式。

补充优化点：增加「重命名文件夹」联动，修改项目名称后，可选是否同步重命名本地项目文件夹（避免路径混乱）；修改操作支持「撤销」（10分钟内），撤销按钮为简约线条图标，贴合黑白灰配色，降低误操作风险。

#### 3. 项目删除

详细设计：支持软删除（标记删除状态，可恢复）/ 硬删除（彻底删除，不可恢复）；硬删除逻辑为弹窗二次确认（提示「删除后将同步删除本地文件夹，是否确认？」，弹窗为黑白灰样式），先删除本地文件夹及所有子文件，再删除程序内项目列表记录，删除按钮为简约线条「×」图标。

补充优化点：增加删除失败处理，如文件夹被占用时，给出「文件正在使用，无法删除」提示（浅灰色文本）并记录日志；增加「删除恢复」，软删除后10分钟内可在回收站恢复，硬删除无恢复功能，回收站图标为简约线条样式。

#### 4. 文件夹监测

详细设计：实时监测为程序运行时，每30秒扫描所有项目文件夹；被动监测为打开项目前校验文件夹是否存在；异常处理为文件夹缺失时，标记项目状态为「文件丢失」（浅灰色标签），并提示用户，提示文本为浅灰色。

补充优化点：监测日志记录文件夹变更时间、原因（程序删除/手动删除），支持查看，日志界面为黑白灰配色；同步规则为文件丢失后，用户可选择「移除项目」或「重新定位文件夹」，操作后同步更新项目状态，操作按钮为简约线条图标。

#### 5. 项目状态管理

详细设计：状态类型包括草稿/进行中/已完成/已归档，均采用灰色系标签（草稿-浅灰、进行中-中灰、已完成-深灰、已归档-灰黑），无鲜艳色；操作方式为点击项目卡片上的状态标签，下拉选择切换，同步更新project.json，下拉菜单为黑白灰样式。

补充优化点：项目列表顶部增加浅灰色筛选栏，可按状态快速筛选项目，筛选图标为简约线条样式；不同状态对应不同灰度标签，视觉区分清晰，贴合整体黑白灰配色。

#### 6. 资源管理器打开

详细设计：点击项目卡片上的「打开文件夹」按钮，直接跳转到该项目的本地文件夹路径，按钮为简约线条文件夹图标，黑白灰配色；同时项目菜单中「打开资源文件管理器」也提供此功能。

补充优化点：增加权限校验，无文件夹访问权限时，给出「无访问权限，请检查文件夹权限设置」的明确提示（浅灰色文本），提升用户体验。

## 2.4 Electron原生菜单集成（补充）

由于本工具基于Electron开发，应用顶部自带原生菜单栏（包含“文件”、“编辑”、“视图”等标准菜单），为保持界面简洁并符合桌面应用习惯，将**系统设置**入口集成至原生菜单中，具体设计如下：

- **文件菜单**：增加「设置」或「偏好设置」项（快捷键可设为Cmd/Ctrl+,），点击打开系统设置面板（同2.3.5所述）。

此设计去除了第四栏标题右侧的「系统设置」齿轮图标，避免界面冗余，同时充分利用原生菜单的通用性，提升用户体验。

1. 新手引导：首次使用时的核心操作指引，指引界面为黑白灰样式，搭配简约线条图标；2. 快捷键支持：新建（Ctrl+N）、保存（Ctrl+S）、删除（Delete）、设置（Cmd/Ctrl+,）等，快捷键提示为浅灰色文本。

# 五、总结

本次优化的核心关键点，一是界面视觉优化，严格遵循黑白灰三色配色，全量使用简约线条图标，开发深浅双主题，确保界面简洁、适配性强；二是布局交互优化，采用横向平铺布局实现项目>片段>镜头>提示词一览无余，将操作按钮集成于各栏标题区域，彻底摒弃顶部全局操作栏，底部显隐面板整合属性操作，无需切换选项卡；三是利用Electron原生菜单集成系统设置，进一步精简界面，符合桌面应用习惯；四是功能细节优化，完善层级联动、批量操作、异常处理等功能，同时移除所有表格，用段落+列表呈现内容，提升文档可读性。

核心优化目标：既保留原始需求中的层级管理和提示词生成核心逻辑，又通过简约视觉设计、流畅交互体验和完善的功能细节，提升产品的可用性、稳定性和扩展性，同时满足用户对界面简洁、配色克制的需求。
