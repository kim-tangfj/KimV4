# 字段自动添加机制说明

## 工作原理

当您在属性表单中修改任意字段并触发自动保存时，系统会：

1. **读取项目文件** - 从 `project.json` 加载当前数据
2. **检查字段** - 对比现有字段和新字段
3. **自动添加** - 如果字段不存在，自动添加并设为空值
4. **保存文件** - 将更新后的数据写入 `project.json`

## 新增字段列表

### 片段属性字段
```javascript
{
  "characters": "",      // 角色描述
  "sceneSetting": "",    // 场景设定
  "musicStyle": "",      // 配乐风格
  "soundEffect": "",     // 音效需求
  "imageRef": "",        // 图片参考
  "videoRef": "",        // 视频参考
  "audioRef": "",        // 音频参考
  "customPrompt": ""     // 补充提示词
}
```

### 镜头属性字段
```javascript
{
  "image": ""            // 分镜图片
}
```

## 使用流程

### 1. 打开旧项目
```
项目列表 → 选择项目 → 打开
```
此时 `project.json` 中没有新字段，但不会报错。

### 2. 选中片段，修改属性
```
片段列表 → 选中片段 → 修改任意字段（如风格、情绪等）
```

### 3. 触发自动保存
- **方式 1**: 点击其他片段（失焦触发）
- **方式 2**: 修改后等待 500ms

### 4. 字段自动添加
系统自动在 `project.json` 中添加：
```json
{
  "shots": [{
    "id": "...",
    "name": "片段 1",
    // 自动添加的字段：
    "characters": "",
    "sceneSetting": "",
    "musicStyle": "",
    "soundEffect": "",
    "imageRef": "",
    "videoRef": "",
    "audioRef": "",
    "customPrompt": ""
  }]
}
```

### 5. 继续编辑
现在所有新字段都可以正常编辑和保存。

## 技术实现

### 片段属性保存逻辑
```javascript
// saveShotProperties 函数
loadResult.projectJson.shots[shotIndex] = {
  ...loadResult.projectJson.shots[shotIndex],  // 保留原有数据
  name: name || '',
  description: description || '',
  // 自动添加缺失字段，如果已存在则使用旧值
  characters: characters !== undefined 
    ? characters 
    : (oldShot.characters || ''),
  sceneSetting: sceneSetting !== undefined 
    ? sceneSetting 
    : (oldShot.sceneSetting || ''),
  // ... 其他字段
};
```

### 镜头属性保存逻辑
```javascript
// saveSceneProperties 函数
shot.scenes[sceneIndex] = {
  ...shot.scenes[sceneIndex],  // 保留原有数据
  name: name || '',
  // 自动添加缺失字段
  image: image !== undefined 
    ? image 
    : (oldScene.image || ''),
  // ... 其他字段
};
```

## 优点

### ✅ 无需手动修改
- 不需要运行脚本
- 不需要手动编辑 JSON 文件
- 打开旧项目即可使用

### ✅ 渐进式更新
- 只更新被编辑的片段/镜头
- 未编辑的片段保持原样
- 不会一次性修改整个项目文件

### ✅ 向后兼容
- 旧项目文件可以正常打开
- 缺失字段自动补全
- 不会丢失任何原有数据

### ✅ 安全可靠
- 使用展开运算符 `...` 保留原有数据
- 使用 `|| ''` 提供默认值
- 使用 `!== undefined` 检查字段是否存在

## 常见问题

### Q: 第一次保存会覆盖原有数据吗？
A: 不会。使用 `...oldShot` 展开运算符保留所有原有数据。

### Q: 未编辑的片段会添加新字段吗？
A: 不会。只更新被编辑的片段/镜头。

### Q: 如何确认字段已添加？
A: 
1. 修改属性后，查看右下角是否显示"已保存"
2. 打开 `work/<项目名>/project.json` 文件
3. 检查是否有新字段

### Q: 如果修改后没有保存怎么办？
A: 检查：
1. 是否在 Electron 环境中运行
2. 项目目录是否正确
3. 控制台是否有错误信息
